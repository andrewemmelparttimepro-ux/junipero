# Legacy Files — Junipero v1

These files are from Junipero v1, the original OpenClaw-based macOS chat client.

**DO NOT USE THESE FILES IN THE CURRENT BUILD.**

They are preserved for reference only. The current app lives in `/Sources/JuniperoApp/`.

## What changed in v2

- OpenClaw backend → Hermes Agent (Nous Research)
- Linen/cream/blue aesthetic → Dark luxury (black + copper)
- 2-panel chat-only → 6-tab console (Command, Threads, Tasks, Review, Approvals, Deliverables)
- Hero clock reskinned with copper/obsidian "JUNIPERO" branding
- Hermes is bundled and auto-managed — users never touch the CLI

## Original structure

- `Sources_v1/` — all SwiftUI source files
- `Package_v1.swift` — original SPM manifest
- `scripts_v1/` — original build/release scripts
- Other docs and configs from the v1 release
